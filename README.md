# BRD4 JQ1 Results Website

Upload the contents of this folder to GitHub.

- `index.html` is the GitHub Pages entry point.
- `results_overview.html` is the same page under its original file name.
- `results/` and `webpage_assets/` contain the local figures and linked tables used by the page.

If using GitHub Pages, set the Pages source to the branch/folder containing this `index.html`.
